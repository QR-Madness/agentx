# Roadmap

Where AgentX is and where it's headed — intentionally high-level. For per-release detail see the
[release notes](https://github.com/QR-Madness/agentx/releases); for live tracking, [issues & PRs](https://github.com/QR-Madness/agentx).

## Shipped

- **Mobile-Ready Alpha** — self-hosted server (auth · production Docker · multi-cluster) + Tauri v2 desktop & Android
- **Layered memory** — episodic / semantic / procedural / working, 5-technique recall, per-agent self-knowledge, background consolidation
- **Agent Alloy** — supervisor + specialist delegation, @-mention routing, per-agent tool isolation
- **Ambassador** — a parallel briefer that narrates a conversation by voice or text, without ever entering the transcript

## Now

- **Ambassador v2** — a conversational, tool-using relay with its own persistent thread
- **Procedural memory** — agents distill and replay "how we work here"
- **Active recall + memory tuning** — closing the query-formulation gap

## Next

- **Memory hardening** — event-sourced write path, bitemporal facts, a capability registry
- **File Workspaces & Document RAG** — hand agents a corpus, not just memory + web
- **Settings Advisor + Manifest** — a self-tuning control plane
- **Factory canvas** — a visual multi-agent workflow editor

> This page only moves when the *direction* does. Version-by-version detail lives in the release notes.
